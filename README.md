**Можливі помилки**

"Файл products.json не знайдено!"

Переконайтеся, що файл products.json знаходиться у папці з виконуваним файлом (У Visual Studio повинен стояти флаг "Copy to Output Directory" = "Copy always")

"Будь ласка, введіть коректну кількість"

Вводьте тільки цілі невід'ємні числа у колонку "Кількість"
